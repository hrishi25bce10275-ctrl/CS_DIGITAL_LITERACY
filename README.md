## Project Overview

Hands-on project building digital literacy skills through five tasks: Canva infographics, professional profiles (GitHub, LinkedIn, Kaggle), coding on HackerRank, Google Forms quiz, communication analysis, and cyber safety reflections. Focuses on content creation, online presence, coding, clear messaging, and phishing prevention in academic scenarios.

## Module Summaries

- **Task 1 - Infographic Design**  
  Canva-based visual on digital literacy: definitions, tools, safety practices, professional presence, email etiquette. Balanced text/icons/colors for engagement.

- **Task 2 - Profile Setup**  
  GitHub for code portfolios; LinkedIn for networking/education; Kaggle for data science/ML practice and competitions.

- **Task 3 - Coding & Quiz**  
  HackerRank Python basics (variables, loops, conditionals); Google Forms quiz with MCQs/short answers on digital literacy.

- **Task 4 - Communication Analysis**  
  Group project mishap from unclear chat deadlines; fixes: structured messages, confirmations, multi-platform sharing.

- **Task 5 - Cyber Safety**  
  Phishing surprises mimicking UPI/bank alerts on public Wi-Fi; new habit: verify via helpline 1930/apps, avoid link clicks.
